# Foundation Services

基础层服务 - 提供核心的认证、用户管理和存储功能。

## 包含的服务

- **AuthService** - 认证、授权、OAuth
- **UsersService** - 用户管理
- **OrganizationsService** - 组织管理
- **TeamsService** - 团队管理
- **StorageService** - 文件存储

## 使用

```typescript
import { 
  AuthService, 
  UsersService, 
  OrganizationsService,
  TeamsService,
  StorageService 
} from '@juanie/service-foundation'
```
