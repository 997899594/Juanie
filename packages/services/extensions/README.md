# Extension Services

扩展层服务 - 提供可选的 AI、监控、通知和安全功能。

## 包含的服务

- **AIService** - AI 助手、Ollama 集成
- **MonitoringService** - 审计日志、成本追踪
- **NotificationsService** - 通知服务
- **SecurityService** - 安全策略

## 使用

```typescript
import { 
  AIService, 
  MonitoringService, 
  NotificationsService,
  SecurityService 
} from '@juanie/service-extensions'
```
