# B站官方设计语言系统 - Bilibili Design System

## 1. 设计哲学

### 1.1 核心理念
**简洁 · 实用 · 社区** - 基于B站官网的真实设计语言
- **简洁直接**：界面干净整洁，信息层次清晰，避免视觉噪音
- **功能优先**：每个设计决策都服务于用户体验和功能实现
- **年轻专业**：既有年轻化的活力，又保持专业的可信度
- **社区导向**：设计支持用户互动和内容创作的社区氛围

### 1.2 设计原则
1. **白色主导**：以纯白色#ffffff为主背景，营造干净专业的视觉环境
2. **粉色点缀**：#fb7299仅用于关键操作和品牌识别，不过度使用
3. **适度圆角**：使用4-6px圆角，既现代又不过分圆润
4. **功能性交互**：交互反馈简洁有效，过渡时间控制在200-300ms
5. **内容为王**：设计服务于内容展示，不喧宾夺主

## 11. 使用指南

### 11.1 颜色使用原则
1. **主色使用**：B站粉色仅用于品牌元素和主要操作
2. **背景色**：以白色和浅灰为主，保持简洁
3. **文字色**：使用灰色系建立清晰层次
4. **功能色**：仅在必要时使用，保持克制

### 11.2 组件使用建议
1. **按钮**：主要操作用粉色，次要操作用描边或灰色
2. **卡片**：使用轻微阴影，避免过度立体效果
3. **输入框**：聚焦时显示粉色边框，保持简洁
4. **标签**：使用淡色背景，避免过于鲜艳

### 11.3 布局原则
1. **留白充足**：给内容足够的呼吸空间
2. **对齐一致**：保持元素对齐的一致性
3. **层次清晰**：通过间距和颜色建立信息层次
4. **响应式**：确保在各种设备上的良好体验

这套设计语言系统严格基于B站官网的实际设计规范，强调简洁、实用和品牌一致性，避免过度装饰，注重用户体验和功能性。
